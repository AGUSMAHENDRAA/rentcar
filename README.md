Car Rental App

Deskripsi Proyek
Aplikasi Car Rental ini dirancang untuk memudahkan pengguna dalam menyewa mobil secara online.  
Pengguna dapat melakukan:
- Registrasi dan login akun.
- Melihat daftar mobil yang tersedia.
- Melihat detail mobil sebelum memesan.
- Mengisi form pemesanan.
- Melihat riwayat pemesanan.
- Mengelola profil pengguna.
- Admin dapat mengelola data mobil dan pesanan.

---

Fitur Utama
1. **Autentikasi Pengguna**
   - Login dan Register menggunakan Firebase Authentication.
2. **Daftar Mobil**
   - Menampilkan semua mobil yang tersedia untuk disewa.
3. **Detail Mobil**
   - Menampilkan informasi lengkap mobil yang dipilih.
4. **Form Pemesanan**
   - Input data pemesanan (tanggal, durasi, dan metode pembayaran).
5. **Riwayat Pemesanan**
   - Melihat pesanan yang sudah dibuat.
6. **Profil Pengguna**
   - Mengedit profil dan melihat informasi akun.
7. **Admin Dashboard**
   - CRUD (Create, Read, Update, Delete) data mobil dan pesanan.


Teknologi yang Digunakan
- **Frontend:** React Native (Expo)
- **Backend:** Firebase Firestore
- **Autentikasi:** Firebase Authentication
- **Penyimpanan Gambar:** Firebase Storage
- **Navigasi:** Expo Router
